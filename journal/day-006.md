# Day 006 - Learning System Restructure

1. What existed before vs. now

Before this restructure, roadmap/progress-tracker.md was doing double duty: it held both my day-by-day completed-class log and the full future curriculum (all the phases/topics you'd eventually cover). There was also no single file telling an AI assistant how to mentor me — teaching philosophy, pacing, docs-first policy, etc. — that context lived only in chat history, which doesn't persist between sessions.

Now there are four distinct files with distinct jobs:

CLAUDE.md (root) — mentor instructions: who am I, how to teach me, lesson structure, documentation policy, pacing rules. Not curriculum content, not progress — behavior instructions for whichever AI is helping.

roadmap/progress-tracker.md — pure history/state: completed days, skills status, current next class. Nothing but facts about what's already happened.

roadmap/curriculum-phases.md — the full future roadmap (Phase 1 foundations through Phase 12 career prep), independent of how far I've actually gotten.

handbook/*.md — split by topic (git.md, linux.md, and more will be added as new topics start) instead of one giant handbook file, so each is a focused reference I can search.

2. Why split it this way

Two different problems were mixed together before: "what's true right now" (progress) versus "what's the plan" (curriculum) versus "how should I be taught" (mentor rules) versus "what do I know" (reference material). Mixing them meant the tracker file kept growing and got harder to scan, and there was no durable place for teaching-style instructions to survive across sessions/tools/machines — that's explicitly why CLAUDE.md exists as a separate file.

3. New mental model — where does X live

Something I did and finished → progress-tracker.md

Something I'll eventually cover → curriculum-phases.md

A reference/how-it-works note on a topic (commands, concepts) → handbook/<topic>.md

A daily narrative (what happened, how I felt, mistakes) → journal/day-XXX.md

How the AI should behave toward me → CLAUDE.md
